# 🚀 React Landing Page (Dockerized)

A modern, responsive landing page built with **React + Vite**, featuring smooth animations and Docker-based deployment.

---

## ✨ Features

* Responsive design (Mobile, Tablet, Desktop)
* Smooth scroll animations
* Interactive contact modal
* Reusable components
* Dockerized with Nginx (production-ready)

---

## 🛠️ Tech Stack

* React + Vite
* Vanilla CSS
* Docker + Nginx

---

## 🚀 Run Locally

```bash
npm install
npm run dev
```

Open: http://localhost:5173

---

## 🐳 Run with Docker

```bash
docker build -t react-landing-page .
docker run -p 8080:80 react-landing-page
```

Open: http://localhost:8080
